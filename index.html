RESUME

Butter Budget
Butter-Budget is a simple browser-based budgeting application. Users can create their own profile where they can enter their budget then their expenses. Butter-Budget has several different expense categories to choose from and displays a user-friendly bar graph to visually display how much they’re spending and how much is left to spend/save. A user’s information is stored within the browser's database, similar to cookies. Butter-Budget was created using three different programming languages, Haskell, Prolog, and JS, that were part of three different programming paradigms.

Word Search Mania
Word Search Mania is a word search game created using the Python 3 and PyQt5 framework. Users can play the game using words randomly pulled from the provided dictionary, or they can play the game using words they input themselves. Players select letters by clicking on them. If the letters are connected horizontally, vertically, or diagonally and match a word in the word bank, then it is highlighted green and the word is crossed off. This project was created to demonstrate our object-oriented programming skills as well as our understanding of Python.
